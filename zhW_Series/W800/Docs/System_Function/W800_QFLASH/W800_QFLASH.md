## 3 FAQ

### 3.1 为什么布局划分总是以 64KB 为倍数划分？

考虑到 QFlash 的实现方式不同，针对掉电过程中，如果有 QFlash 的写动作，可能会破坏整个扇区的操作。

### 3.2 为什么总是留一部分升级区在内部 Flash 里？

因为 ROM 里总是认定 SECBOOT 的升级区域在内部 QFlash 里。
为了预防 SECBOOT 升级，必须把 OTA 区域预留在内部 QFlash 里。
